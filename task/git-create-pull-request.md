# ドラフト状態のプルリクエストを作成するタスク

# 手順
1. 以下の情報が提供されていないければ、ユーザーに質問して提供してもらう
    - PRテンプレートのパス
    - PRマージ先のブランチ
2. PRテンプレートをReadツールで読み取ってください
3. mcp-github__create_pull_request を使い、PRテンプレートを使ってDraftPRを作成してください
    - ドラフト状態で作成すること
    - タイトルは空コミットのコミットメッセージをそのまま使用
    - PRの説明にある`#xxxxx`をRedmineチケットのリンクに置き換えること
4. mcp-redmine__redmine_add_comment を使い、以下のコメントをチケットに追加する
    '''
    h3. PR

    手順3で作成したPRのURL
    '''
