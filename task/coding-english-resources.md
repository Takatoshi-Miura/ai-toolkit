# スプレッドシート記載の英語文言をアプリの文言リソースに追加するタスク

# 手順
1. 以下の情報が提供されていなければユーザーに質問して提供してもらう
    - スプレッドシートURLと読み取るシート名（必須）
    - 追加指示（任意）
2. /read-google-drive-skill でスプレッドシートを読み取る
3. プロジェクト内を検索し、英語文言リソース、日本語文言リソースファイルを見つける
4. 「文言追加ルール」に従い、英語文言リソースファイルに2で読み取った英語文言を追加する

# 文言追加ルール
- シートに記載されている文言のみ追加すること（日本語リソースのみ存在する文言があっても問題ない）
- 追加する位置は日本語文言リソースの並び順と合わせること
