# 現在のブランチを強制プッシュするタスク

## 概要
スカッシュやリベース後など、リモートの履歴を上書きする必要がある場合に、現在のブランチを強制プッシュする。

## 手順

### Step 1: 現在のブランチ名を取得
1. 以下のコマンドで現在のブランチ名を取得する
   ```bash
   git branch --show-current
   ```
2. 取得したブランチ名をユーザーに表示する
   ```
   現在のブランチ: [ブランチ名]
   ```

### Step 2: 実行確認
1. 以下の確認メッセージを表示し、ユーザーの承認を得る
   ```
   以下のコマンドを実行します:
   git push --force origin [ブランチ名]

   続行しますか？
   ```
2. ユーザーが承認した場合のみ Step 3 へ進む

### Step 3: 強制プッシュの実行
1. 以下のコマンドを実行する
   ```bash
   git push --force origin [ブランチ名]
   ```

### Step 4: 結果の報告
1. 実行結果を表示する
   - 成功時: `✅ 強制プッシュが完了しました`
   - 失敗時: エラーメッセージを表示
