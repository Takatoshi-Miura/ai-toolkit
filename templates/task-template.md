# タスクファイルのテンプレート

以下のテンプレートを使用してタスクファイルを生成してください。
`{変数名}` の部分を適切な値に置き換えてください。

## テンプレート

```markdown
# {command_name} コマンド用タスク

## 概要
{description}

## 手順
{steps}
```

## 変数の説明

| 変数名 | 説明 | 例 |
|--------|------|-----|
| `{command_name}` | コマンド名（kebab-case） | `review-code`, `analyze-performance` |
| `{description}` | タスクの概要説明 | PRのコードを分析し、レビュー観点に沿って指摘事項をまとめる |
| `{steps}` | 番号付きの手順リスト | 1. xxx<br>2. xxx<br>3. xxx |

## 手順の書き方のポイント

- 番号付きリストで記載する
- 各手順は具体的かつ明確に記載する
- 必要に応じてチェックリストやフォーマットセクションを追加する
