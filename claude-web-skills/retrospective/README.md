# 振り返りスキル - Claude Web版セットアップ

Claude Webのプロジェクト機能で週次振り返りを実行するためのセットアップ手順です。

## 前提条件

- Claude Pro/Team/Enterpriseプラン（プロジェクト機能が必要）
- Googleアカウント（カレンダー・Driveへのアクセス）

## セットアップ手順

### 1. プロジェクトの作成

[Claude.ai](https://claude.ai/) でプロジェクトを作成します。

### 2. コネクタの設定

プロジェクトの設定画面で以下のコネクタを有効化:

| コネクタ | 用途 |
|---------|------|
| Googleカレンダー | LifeGraph分析（活動時間の集計） |
| Google Drive | 日次記録・パーソナルコンテキストの読み取り |

### 3. カスタムスキルの追加

プロジェクトのカスタマイズから、以下のファイルをスキルとしてアップロード:

| ファイル | 役割 |
|---------|------|
| `SKILL.md` | 実行手順のオーケストレーター |
| `REFERENCE-LIFEGRAPH.md` | LifeGraph分析の取得・分析仕様 |
| `REFERENCE-DAILY.md` | 日次記録の取得・分析仕様 |
| `REFERENCE-SUMMARY.md` | 目標提案・総評の仕様 |

## 使い方

プロジェクト内の会話で「振り返りお願い」「週次振り返り」などと依頼するだけです。
SKILL.mdの手順に従ってデータ取得・分析・レポート出力が実行されます。

## Claude Code版との違い

| 項目 | Claude Web版 | Claude Code版 |
|------|-------------|---------------|
| 対応期間 | 週次のみ | 週次・月次 |
| 金銭分析 | 非対応（スプレッドシート読み取り不可） | 対応 |
| LifeGraph取得 | Googleカレンダーコネクタ（生データ） | Pythonスクリプト（集計スプレッドシート） |
| 日次記録取得 | Google Driveコネクタ | Pythonスクリプト |
| 出力形式 | 会話内Markdown | ファイル出力（~/Downloads/） |
| 会話履歴 | プロジェクト内に蓄積 | セッションごとにリセット |
