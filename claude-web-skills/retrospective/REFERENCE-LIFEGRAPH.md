# LifeGraph分析リファレンス（Claude Web版）

対象: 週次

## データ取得

Googleカレンダーコネクタを使用して、対象期間（直近1週間）のカレンダーイベントを取得する。

### カレンダータグ定義

イベントには以下のハッシュタグが付与されている:

| タグ | カテゴリ名 | 説明 |
|------|-----------|------|
| #work | 仕事 | 業務時間 |
| #life | 生活 | 日常生活（家事・食事等） |
| #sleep | 睡眠 | 睡眠時間 |
| #undo | 運動 | 運動・フィットネス |
| #idea | 思考整理 | 思考整理・内省 |
| #ref | リフレッシュ | 休息・気分転換 |
| #douga | YouTube活動 | YouTube関連の活動 |
| #skill | 自己研鑽 | 学習・スキルアップ |
| #book | 読書 | 読書 |
| #code | プログラミング | プログラミング |
| #chi | ちーちゃん | パートナーとの時間 |

### 集計ルール

- **自己投資**: #skill + #book + #code の合算
- **未登録時間**: 1日24hのうち、イベントが登録されていない時間帯は #sleep（睡眠）として扱う

取得したイベントデータから以下を集計:
- 各タグの活動時間
- 日別の活動時間の推移

**備考:**
- Claude Code版ではスプレッドシートの集計データ（パフォーマンス点数等）を使用するが、Claude Web版ではカレンダーの生イベントデータから直接分析する
- パフォーマンス点数はカレンダーから取得不可のため、活動時間ベースの分析に変更
- パートナーとの時間計測は2025年6月から開始

## 分析観点

- 週次トレンド（#work, #sleep, 自己投資, #undo, #chi の推移を表形式で表示）
- 活動バランス（#work, 自己投資, #sleep, #life, #ref 等の時間配分）
- 自己投資の状況（#skill + #book + #code の合計時間と内訳バランス）
- パートナーとの時間（#chi の平日/休日別の平均時間）
- その他の活動（#undo, #idea, #douga, #ref など個別タグの傾向を簡潔に分析）

## 出力フォーマット

```markdown
## LifeGraph分析

### 週次トレンド
| 日付 | 仕事(#work) | 睡眠(#sleep) | 自己投資 | 運動(#undo) | パートナー(#chi) |
|------|------------|-------------|---------|------------|----------------|
| MM/DD(曜) | X.Xh | X.Xh | X.Xh | X.Xh | X.Xh |
（傾向を1-2行で簡潔に記述）

### 活動バランス
（各タグの時間配分を分析し、2-3行で簡潔に記述）

### 自己投資の状況
（#skill / #book / #code の内訳と合計時間、平日・休日の傾向を1-2行で簡潔に記述）

### パートナーとの時間
（#chi の平日・休日の平均時間、一般的な水準との比較を1-2行で簡潔に記述）

### その他の活動
- **運動(#undo)**: （頻度と傾向を1行で）
- **思考整理(#idea)**: （増減傾向や効果を1行で）
- **リフレッシュ(#ref)**: （傾向を1行で）
- **YouTube活動(#douga)**: （傾向を1行で）
- **その他気づき**: （その他の傾向を自由に記述）
```
