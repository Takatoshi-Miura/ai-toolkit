# LifeGraph分析リファレンス

対象: 週次・月次共通

## データ取得

### Step 1: カレンダーイベント取得

```bash
python3 ~/.claude/skills/retrospective/scripts/read_calendar.py --start YYYY-MM-DD --end YYYY-MM-DD
```

**出力形式:** JSON。`events`にイベント一覧、`dailySummary`に日別・時間帯別の集約データが含まれる。

**タグによるカテゴリ分類:** イベントタイトルの`#tag`（例: `#work`, `#sleep`）からカテゴリを自動判定。複数タグは全カテゴリにフル計上。

### Step 2: スプレッドシート補完データ取得

```bash
python3 ~/.claude/skills/retrospective/scripts/read_lifegraph.py
```

**取得対象:** カレンダーでは取得できないデータ（歩数、コード変更行数等）。

**読み取り範囲:** CONFIG.mdで設定（デフォルト: A~P列）

**備考:**
- パートナーとの時間計測は2025年6月から開始
- N列: 歩数（2026年3月追加、既存N列以降は右にシフト）

## 分析観点

- 時間帯別の行動パターン（早朝/午前/午後/夕方/夜の各時間帯における行動とルーティン傾向）
- 週次トレンド（パフォーマンス、主要活動の推移を表形式で表示）
- パフォーマンスに影響を及ぼしている要因（高パフォーマンス日の共通点を簡潔に）
- 活動バランス（仕事、自己投資、睡眠、生活、リフレッシュ等の時間配分）
- 自己投資の状況（自己研鑽+読書+プログラミングの合計時間とバランス）
- パートナーとの時間（平日/休日別の平均時間）
- その他の活動（運動、思考整理、リフレッシュ、YouTube活動を個別に分析）

## 出力フォーマット

