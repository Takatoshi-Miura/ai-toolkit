# LifeGraph分析リファレンス

対象: 週次・月次共通

## データ取得

```bash
SCRIPT_PATH="~/.claude/skills/retrospective/scripts/read_drive_file.py"

python3 $SCRIPT_PATH 1WF58VNM0lGfN-YKqR2ySXU_EKQQCpyrV4da8WiVtoBo sheets "Googleカレンダー集計"
```

**出力形式:** JSON。`content` フィールドにデータが含まれる。

**読み取り範囲:** A~O列

**備考:** パートナーとの時間計測は2025年6月から開始

## 分析観点

- 週次トレンド（パフォーマンス、主要活動の推移を表形式で表示）
- パフォーマンスに影響を及ぼしている要因（高パフォーマンス日の共通点を簡潔に）
- 自己投資の状況（自己研鑽+読書+プログラミングの合計時間とバランス）
- パートナーとの時間（平日/休日別の平均時間）
- その他の活動（運動、思考整理など未活用データを簡潔に分析）
