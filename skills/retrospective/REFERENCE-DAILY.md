# 日次記録分析リファレンス

対象: 週次・月次共通

## データ取得

```bash
python3 ~/.claude/skills/retrospective/scripts/read_daily.py
```

**出力形式:** JSON。`goalsTab` と `monthTab` フィールドにそれぞれのタブのテキストが含まれる。

**読み取り範囲:** 「今年の目標」タブとCONFIG.mdの `month_tab` で指定された月タブ

## 分析観点

- 各カテゴリの目標に対して達成できたこと・貢献したアクションを複数抽出
- 振り返り期間を通じて気づいた傾向や発見を複数整理
- 目標達成に向けて今後取り組めるアクションを複数提案

## 出力フォーマット（週次）

```markdown
## 日々の振り返り分析

### 目標達成のために今週できたこと
#### 仕事
- （できたこと1）
- （できたこと2）
#### 人間関係
- （できたこと1）
#### 金銭
- （できたこと1）
#### 健康・メンタル
- （できたこと1）

### 傾向分析や発見
| カテゴリ | 傾向・発見 |
|---------|----------|
| 仕事 | （傾向や発見1） |
| 人間関係 | （傾向や発見1） |
| 金銭 | （傾向や発見1） |
| 健康・メンタル | （傾向や発見1） |

### 目標達成のために来週できること
| カテゴリ | アクション |
|---------|----------|
| 仕事 | （今後できること1） |
| 仕事 | （今後できること2） |
| 人間関係 | （今後できること1） |
| 金銭 | （今後できること1） |
| 健康・メンタル | （今後できること1） |
```

※月次の場合は「今週」→「今月」、「来週」→「来月」に変更
