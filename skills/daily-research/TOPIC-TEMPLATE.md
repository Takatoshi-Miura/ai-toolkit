# トピック追加テンプレート

新しいトピックを追加する際は、以下をコピーしてTOPICS.mdに貼り付け、各項目を編集してください。

---

## N. {トピック名}

| 項目 | 値 |
|------|-----|
| カテゴリ | {tech / news / economy / etc.} |
| 検索クエリ | `{WebSearchに渡す検索文字列}` |
| 取得件数 | {3-10の数値} |
| 直接取得URL | {WebFetchで直接取得するURL。不要なら `-`} |
| 言語 | {ja / en} |
| 優先度 | {high / medium / low} |

**補足**: {任意の補足説明。不要なら削除}

### 出力形式

```
### {番号}. {記事タイトル}
- **ソース**: {ソースサイト名} | [リンク]({URL})
- **解説**: {記事の解説}
```

{出力形式の補足指示があれば記載。例: 「リンク先を見なくても内容が把握できるレベルで解説を記載する」}

---

## フィールド説明

| フィールド | 必須 | 説明 |
|-----------|------|------|
| カテゴリ | Yes | 分類タグ（tech, news, economy など自由に設定可） |
| 検索クエリ | Yes | WebSearchに渡す検索文字列 |
| 取得件数 | Yes | 検索結果から取得する記事数（推奨: 3-10） |
| 直接取得URL | No | WebFetchで直接取得するURL（不要なら `-` を記入） |
| 言語 | Yes | 検索言語（ja, en） |
| 優先度 | Yes | high / medium / low（収集失敗時のリトライ判断に使用） |
| 出力形式 | Yes | そのトピックのmd出力テンプレート。自由にカスタマイズ可能 |
