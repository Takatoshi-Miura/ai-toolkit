# スケジュール表

実行時刻と紐づけるテーマIDを記載する。
Claude は起動時刻と照合し、0〜15 分以内に一致する行のテーマを実行する。
一致なしの場合は何もせず終了する。

| 時刻  | テーマID | 説明               |
|-------|----------|--------------------|
| 07:00 | outing   | 週末お出かけ提案   |
| 08:00 | book     | 週次読書推薦       |

---

# テーマ: outing

## 宛先

- groupId: C1234567890abcdef...

## 調査条件

- エリア: 蓮田駅(埼玉県)から片道 1 時間以内
- 移動手段: 車で行ける場所を優先。車アクセスが良いスポットを上位に扱う
- ジャンル: 屋内外どちらも可。美術館・公園・カフェ巡り・自然散策が好み
- 避けたいもの: 強い混雑、行列必須の人気スポット、派手なテーマパーク、駐車場が確保できない場所
- 同行者: 2 人(自分を含む)
- 対象日: 次の土曜日。雨天なら屋内中心に切り替える
- 除外: outing/history.md に含まれる場所はすべて選ばない

## メッセージフォーマット

以下のテンプレートに沿って本文を組み立てる。各行は必須。絵文字は 1 行目のみ可。

🗓 {対象日(YYYY/MM/DD (曜))}
📍 {スポット名}({市町村名 / 蓮田から車で約 X 分})
⏱ 片道目安 {所要時間}
💡 {おすすめ理由を 2〜3 行}
🚗 車: {所要時間・駐車場の有無や料金}
🔗 公式: {公式 URL}
🗺 地図: {Google Maps URL}

# テーマ: book

## 宛先

- groupId: C1234567890abcdef...

## 調査条件

- ジャンル: ビジネス書・技術書・自己啓発
- 対象: 直近 1〜2 年以内に出版された比較的新しい本を優先
- 避けたいもの: book/history.md に含まれる本
- 推薦基準: 実践的な内容、評価が高いもの

## メッセージフォーマット

以下のテンプレートに沿って本文を組み立てる。絵文字は 1 行目のみ可。

📚 今週のおすすめ本
{タイトル} / {著者}({出版年})
💡 {推薦理由を 2〜3 行}
🔗 {Amazon または出版社 URL}
